# TEK1314-2025-Kel08-KelasF2
# PBL Network Security - Kelompok 8

Pada Design Phase, kelompok kami merancang topologi jaringan untuk simulasi serangan Malware Reverse TCP dalam satu subnet 192.168.8.0/24. Skenario yang dipilih adalah endpoint Windows 11 sebagai target yang menjalankan Trojan dan melakukan koneksi Reverse TCP ke Kali Linux sebagai attacker, sementara seluruh aktivitas jaringan dimonitor menggunakan Security Onion. Perancangan ini dilakukan untuk mempersiapkan fase implementasi dan simulasi serangan pada minggu berikutnya.
